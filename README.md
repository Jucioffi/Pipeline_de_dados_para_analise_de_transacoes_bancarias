# Implementa-o_de_Pipeline_de-Dados_para_An-lise_de_Transa-es_Banc-rias
Desenvolvimento de um MVP de engenharia de dados aplicado a transações bancárias, com ingestão em nuvem, pipeline Bronze–Silver–Gold, modelagem dimensional e análises analíticas. O projeto utiliza Databricks e Delta Lake para garantir rastreabilidade e reprodutibilidade.
